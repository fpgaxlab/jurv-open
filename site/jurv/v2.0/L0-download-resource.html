下载实验材料

实验材料发布在gitee开源项目托管网站的openJURV项目。建议在电脑中安装Git，以方便获得后续的更新。如果暂时不想安装Git，也可以直接登录Gitee下载。下面分别介绍两种方法。

使用Git获取实验材料

  1. 安装Git

    下载并安装 Git for Windows。 如果遇到问题可阅读官方文档。

  2. 打开GitBash终端

    在想要存放实验材料的文件夹上右键点击 Git Bash Here ，打开GitBash终端。或者从Windows开始菜单启动GitBash，用 cd 命令进入想要存放实验材料的文件夹。

  3. 下载实验材料

    在GitBash终端中执行下面的命令。

    git clone https://gitee.com/fpga-lab/jurv-open.git

  4. 更新实验材料

    以后如果需要获取最后的更新，在GitBash终端中执行下面的命令进入jurv-open文件夹后拉取更新的部分。

    cd jurv-open
git pull

登录gitee网站下载实验材料

  1. 注册为gitee用户并登录；

  2. 转到openJURV项目，点击 克隆/下载 按钮，再点击 下载ZIP 。

  3. 如果以后网站上有更新，重复上述操作。

这种方法的“优点”是不用安装Git，缺点是每次都要下载完整的项目，速度稍慢。
