安装软件

安装FPGA设计软件

Quartus Prime分为Pro、Standard 和 Lite Edition，其中 Lite（精简版）是免费的，完全能够满足教学使用，因此建议安装Lite Edition。软件可从Intel官方网站下载；本书使用Cyclone IV E系列的器件，所以还需要下载Cyclone IV Device Support Package。安装时只要将下载的器件库和Quartus安装程序放在同一个文件夹，安装程序会自动搜寻器件库并默认选择所包含的器件系列。本书使用的Quartus Prime版本为20.1.1，也可以使用更新版本的Quartus设计软件，支持Cyclone IV E器件即可，只是在界面菜单操作等细节上可能与本书不完全相同。

安装时采用软件的默认设置即可。如果要改变安装路径，注意安装路径不要有汉字和空格。

安装本地实验调试软件

本地实验软件有两种：WeLab和JULAB。请向指导教师咨询是否需要使用本地实验软件以及使用哪一种。

实验软件发布在gitee开源项目托管网站中openJURV项目的software文件夹，从中下载需要使用的实验调试软件。WeLab和JULAB均无需安装，解压后即可使用。

点击本电子书网站右上角的 Download 按钮可跳转到openJURV项目首页。
